# C++ Function Programs 💻

This repository contains 7 basic C++ programs based on Functions.  
All programs are beginner friendly and well commented.

## 📚 What is a Function?
A function is a block of code that runs only when it is called.  
We use functions to avoid repeating code and to make our program organized.

## ✅ Key Benefits
- **Reusability** - Write once, use many times
- **Modularity** - Break big problems into small parts  
- **Easy Debugging** - Find and fix errors easily

| 6 | Find Maximum | Find max between two numbers |
| 7 | Factorial | Calculate factorial of a number |

# C++ Array Programs 📊

This repository contains 7 basic C++ programs based on Arrays.  
All programs include input, logic, and output with proper comments.

## 📚 What is an Array?
An array is a collection of elements of the same data type stored in contiguous memory locations.  
We use arrays to store multiple values in a single variable.

## ✅ Key Benefits
- **Store Multiple Values** - One variable, many data
- **Easy Access** - Access elements using index
- **Loop Friendly** - Process many elements with loops
